# Dotfiles

This is my dot files on a Gentoo Linux laptop, with SwayWM and pipewire.

# Software that this dotfile includes

| Name | Description |
|------|-------------|
|swayWM|Tiling WM|
|waybar|Custom bar for sway|
|swayNC|Notification center|
|wofi|App launcher|
|kitty|Terminal emulator|
|mpv|Video player|
|tmux|Split terminal|
|zathura|PDF viewer|
